# Deepfake Detection System

A Deepfake Detection web application built using Flask and Deep Learning models.

## Features
- Detects fake and real images/videos
- Upload image and video files
- Deep learning based prediction
- User-friendly web interface

## Technologies Used
- Python
- Flask
- TensorFlow / PyTorch
- HTML, CSS, JavaScript

## Project Structure
- backend/
- templates/
- static/
- dataset/

## How to Run

```bash
pip install -r requirements.txt
python app.py
```
